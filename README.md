# Corporate Mobile App

## Description
Mobile application prototype for automating corporate HR services and internal communications.

## Release 1 Scope
- Authentication
- User profile (basic)
- Notifications
- Help & Contacts
- Settings (language, theme)

## Platforms
- iOS 16+
- Android 12+

## UX Prototype
Wireframes created in Miro.

## Future Releases
Release 2 and Release 3 will extend functionality.
